Upload Resume

Educational exploration for the democratisation of Job-Recommendations

Welcome, automated job-recommendations (like many many other programs) have always been an exclusive tool of private companies. However, with the recent advent of FOSS campaign, everyday consumers are now rising to be creators of programs too. This is largely due to a plethora of educational materials such as the freecodecamp and also software like python. This app attempts to use out-of-the-box and easily accessible python libraries to 1) automate resume processing and 2) searching of relevant jobs via major job-sites. In doing so, we will be able to explore how much easier it is for consumers to create sophisticated programs which were once exclusive.
This app uses two capabilities of python to conduct automation, namely Machine Learning and Webscraping.
  • pyresparser, a Machine Learning library provided by Mr Omkar Pathak, for resume processing.
  • chromedriver, selenium, request and bs4 libraries for webscraping of jobsites.
This app is strictly for educational purposes. The websites scraped are linkedin, indeed, mycareersfuture.gov.sg.
